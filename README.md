# やることリスト (Todo List App)

## 概要
シンプルなタスク管理アプリ。日常のやることを簡単に管理できます。

## スクリーンショット
![App Screenshot](screenshots/screenshot1.png)

## 機能
- タスクの追加
- タスクの完了/未完了の切り替え
- タスクの削除（スワイプまたはボタン）
- 完了数の統計表示
- データの永続化（アプリを閉じても保存）

## 使用技術
- **言語**: Dart
- **フレームワーク**: Flutter
- **状態管理**: setState
- **データ保存**: SharedPreferences

## 開発環境
- Flutter SDK: 3.5.0+
- Dart SDK: 3.5.0+

## インストールと実行

### 前提条件
- Flutter SDK がインストールされていること
- Android Studio または Xcode がインストールされていること

### 実行方法
```bash
# 依存関係をインストール
flutter pub get

# アプリを実行
flutter run
```
